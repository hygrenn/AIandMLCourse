# Week 3: 신경망 기초 — PySide6 GUI

## 실행 방법

```bash
cd gui_app
uv run python main_gui.py
```

---

## 탭 구성 및 기능

### 탭 1 — 퍼셉트론
- 학습률, 에폭 수 설정 후 **학습 시작** 버튼 클릭
- AND / OR / XOR 결정 경계 그래프 표시
- 하단에 각 게이트 예측 결과 및 오류 개수 표시

### 탭 2 — 활성화 함수
- 앱 실행 시 자동 렌더링 (조작 없음)
- Sigmoid / Tanh / ReLU / Leaky ReLU 함수 및 미분값 비교 (2×2 그래프)

### 탭 3 — 순전파
- 앱 실행 시 자동 렌더링 (조작 없음)
- 2-3-1 네트워크의 Layer별 값 변화 시각화 (네트워크 구조 다이어그램 포함)

### 탭 4 — MLP / XOR
- 은닉층 뉴런 수, 학습률, 에폭 수 설정 후 **학습 시작** 버튼 클릭
- 백그라운드 학습 (진행률 바 표시, UI 블로킹 없음)
- 손실 곡선 / XOR 결정 경계 / 은닉층 활성화 그래프 표시

### 탭 5 — 보편 근사
- 활성화 함수(tanh / relu / sigmoid), 에폭 수 설정 후 **학습 시작** 버튼 클릭
- 9개 모델(함수 3종 × 뉴런 수 3종) 순차 학습 (약 20~40초)
- 3×3 그리드 그래프 및 MSE 비교표 표시

- 저장 위치: `gui_app/outputs/` (자동 생성)
